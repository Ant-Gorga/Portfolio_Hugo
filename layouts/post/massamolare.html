#Descrizione

Il programma é diviso in funzioni, poiché é sfruttato anche come modulo nella versione con Tk, quando invece viene chiamato direttamente, il file si comporta di conseguenza, grazie a questa istruzione. 

      
  if __name__ == "__main__":
    try:
      if sys.argv[1] == "-h":
        printhelp()
      else:
        ris = MassaMolare(sys.argv[1])
        if ris != ValueError:
          print("La Massa Molare di " + sys.argv[1] + " é:"+ str(ris))
        else:
          print("Ricontrolla la formula!")
    except IndexError:
      printhelp()

La variabile __name__ é una variabile speciale settata da python ad ogni esecuzione, se il programma é eseguito direttamente, questa prenderá come valore __main__ e di conseguenza verrá eseguito il codice che permette il funzionamento del programma. Diversamente se viene imporatto come modulo, la variabile avrá valore MassaMolare, saltando quindi il blocco di codice.

La logica del programma é richiamata dalla funzione btncliccato. Con la funzione mainloop, chiamata sull'oggetto di root windows fa renderizzare la finestra e iniziare il loop del programma.
